# Smart Office 

Sistema de gestão empresarial inteligente desenvolvido com Python, FastAPI, PostgreSQL e Inteligência Artificial.

## Objetivo

O Smart Office tem como objetivo centralizar processos empresariais, oferecendo:

- Gestão de clientes
- Controle de produtos
- Gestão de funcionários
- Controle de vendas
- Relatórios automatizados
- Assistente com Inteligência Artificial

## Tecnologias

### Backend
- Python
- FastAPI
- SQLAlchemy
- PostgreSQL

### Frontend
- React
- TypeScript

### Outros
- Docker
- Git
- OpenAI API
